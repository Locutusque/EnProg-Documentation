About EnProg

About EnProg Java Variant

The EnProg Java Variant is a Java implementation of the EnProg programming language. It was created by Locutusque and is hosted on GitHub.

This implementation aims to provide a fast and efficient way to execute EnProg code on the Java Virtual Machine (JVM).

If you're interested in learning more about EnProg, check out the documentation or download the Java Variant to try it out for yourself!

About EnProg Python Variant

The EnProg Python Variant is a Python implementation of the EnProg programming language. It was created by Locutusque and is hosted on GitHub.

This implementation of EnProg is designed to be easy to use and to provide a straightforward way to learn the language. It is written in Python 3.9 and can be installed on GitHub

If you are interested in contributing to the EnProg Python Variant project, please visit the GitHub repository.
